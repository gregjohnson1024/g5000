# G5000 Chart Plotter + AIS Overlay + CPA Alarm — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a simple top-down chart plotter at `/chart` that shows own-boat plus surrounding AIS targets, with closest-point-of-approach (CPA) and time-to-CPA (TCPA) computed per target, and a configurable alarm when both go below safe thresholds.

**Architecture:** AIS targets are tracked in a `globalThis`-backed singleton registry (same pattern as `DeviceRegistry` and the source-mode controller). The bridge's channel mapper decodes incoming AIS PGNs (129038, 129039, 129040, 129809, 129810) and feeds the registry. A small compute pipeline subscribes to the registry plus own-boat nav data, computes CPA/TCPA per target periodically, and publishes `alarm.ais.cpa` events when thresholds are breached. The web UI exposes a `/chart` SVG canvas page and a `/api/ais/targets` REST endpoint.

**Tech Stack:** Existing — TypeScript, RxJS, Next.js 16, Tailwind, vitest. Re-uses canboatjs's built-in AIS PGN decoders.

#### Step 2: Implementation

```ts
// packages/bridge/src/ais/targets-registry.ts
import {
  setSharedAisTargets,
  getSharedAisTargets,
  type AisTarget,
  type AisTargetsRegistry,
} from '@g5000/core';

export function createAisTargetsRegistry(): AisTargetsRegistry {
  const existing = getSharedAisTargets();
  if (existing) return existing;

  const byMmsi = new Map<number, AisTarget>();
  const registry: AisTargetsRegistry = {
    all: () => Array.from(byMmsi.values()),
    get: (mmsi) => byMmsi.get(mmsi),
    upsert: (u) => {
      const prev = byMmsi.get(u.mmsi);
      byMmsi.set(u.mmsi, { ...prev, ...u, lastSeenMs: Date.now() } as AisTarget);
    },
    evictStale: (maxAgeMs) => {
      const cutoff = Date.now() - maxAgeMs;
      let dropped = 0;
      for (const [mmsi, t] of byMmsi) {
        if (t.lastSeenMs < cutoff) {
          byMmsi.delete(mmsi);
          dropped += 1;
        }
      }
      return dropped;
    },
    clear: () => byMmsi.clear(),
  };
  setSharedAisTargets(registry);
  return registry;
}
```

#### Step 1: Add an AIS handler in the channel-mapper

Read `packages/bridge/src/channel-mapper.ts` to understand the existing dispatch structure. Add a new branch for the AIS PGNs that does NOT publish to the bus (we keep the registry out of the bus to avoid per-MMSI channel proliferation). The handler should:

1. Look up `getSharedAisTargets()` (lazily create via `createAisTargetsRegistry()` if missing).
2. Map the decoded fields into `Partial<AisTarget>`.
3. Call `registry.upsert({ mmsi, vesselClass, ...fields })`.

```ts
// inside channel-mapper.ts, or a sibling file ais-handler.ts:
import { getSharedAisTargets } from '@g5000/core';
import { createAisTargetsRegistry } from './ais/targets-registry.js';

const AIS_PGNS = new Set([129038, 129039, 129040, 129794, 129809, 129810]);

export function handleAisPgn(pgn: number, fields: Record<string, unknown>): void {
  if (!AIS_PGNS.has(pgn)) return;
  const registry = getSharedAisTargets() ?? createAisTargetsRegistry();
  const mmsi = Number(fields['User ID'] ?? fields['userId'] ?? fields['MMSI'] ?? NaN);
  if (!Number.isFinite(mmsi) || mmsi <= 0) return;

  const vesselClass: 'A' | 'B' | 'unknown' =
    pgn === 129038 || pgn === 129794
      ? 'A'
      : pgn === 129039 || pgn === 129040 || pgn === 129809 || pgn === 129810
        ? 'B'
        : 'unknown';

  const update: Parameters<typeof registry.upsert>[0] = { mmsi, vesselClass };
  const lat = Number(fields['Latitude'] ?? fields['latitude']);
  const lon = Number(fields['Longitude'] ?? fields['longitude']);
  if (Number.isFinite(lat)) update.lat = lat;
  if (Number.isFinite(lon)) update.lon = lon;
  const cog = Number(fields['COG'] ?? fields['cog']);
  if (Number.isFinite(cog)) update.cog = cog;
  const sog = Number(fields['SOG'] ?? fields['sog']);
  if (Number.isFinite(sog)) update.sog = sog;
  const heading = Number(fields['Heading'] ?? fields['heading']);
  if (Number.isFinite(heading)) update.heading = heading;
  const rot = Number(fields['Rate of Turn'] ?? fields['rateOfTurn']);
  if (Number.isFinite(rot)) update.rateOfTurn = rot;
  const name = typeof fields['Name'] === 'string' ? (fields['Name'] as string).trim() : undefined;
  if (name) update.name = name;
  const length = Number(fields['Length'] ?? fields['length']);
  if (Number.isFinite(length)) update.length = length;
  const beam = Number(fields['Beam'] ?? fields['beam']);
  if (Number.isFinite(beam)) update.beam = beam;
  const vesselType = Number(fields['Type of Ship'] ?? fields['vesselType']);
  if (Number.isFinite(vesselType)) update.vesselType = vesselType;

  registry.upsert(update);
}
```

Wire this into the existing PGN dispatch in `channel-mapper.ts`: when a decoded PGN's pgn number is in `AIS_PGNS`, call `handleAisPgn(pgn, fields)` INSTEAD of (or in addition to) the normal channel-publish path. Don't double-decode.

#### Math

Given own (lat, lon, cog, sog) and target (lat, lon, cog, sog), all in SI (radians, m/s):

1. Convert to local x/y around own boat using equirectangular projection. For small ranges (< 100 nm) the distortion is negligible.

   ```
   metersPerDegLat = 111_320            // approx, constant
   metersPerDegLon = 111_320 * cos(own.lat_radians)
   target.x = (target.lon - own.lon) * metersPerDegLon
   target.y = (target.lat - own.lat) * metersPerDegLat
   own.x = own.y = 0
   ```

   Use _degrees_ for lat/lon in the input (matches our `AisTarget.lat/lon` definition).

2. Velocity vectors (compass-bearing to math-vector: x=east is sog*sin(cog), y=north is sog*cos(cog)):

   ```
   own.vx  = own.sog  * sin(own.cog)
   own.vy  = own.sog  * cos(own.cog)
   tgt.vx  = tgt.sog  * sin(tgt.cog)
   tgt.vy  = tgt.sog  * cos(tgt.cog)
   ```

3. Relative position and velocity (target relative to own):

   ```
   rx = target.x   (own.x = 0)
   ry = target.y
   rvx = tgt.vx - own.vx
   rvy = tgt.vy - own.vy
   ```

4. TCPA — time at which (rx + tcpa*rvx, ry + tcpa*rvy) is minimum:

   ```
   |rvel|² = rvx² + rvy²
   if |rvel|² == 0:
     tcpa = 0    (no relative motion; CPA = current distance)
   else:
     tcpa = -(rx*rvx + ry*rvy) / |rvel|²
   ```

   Negative tcpa means the closest point is already in the past — boats are diverging.

5. CPA — minimum distance:
   ```
   cpa_x = rx + tcpa * rvx
   cpa_y = ry + tcpa * rvy
   cpa = sqrt(cpa_x² + cpa_y²)
   ```

#### Implementation

```ts
// packages/compute/src/ais/cpa.ts
export interface CpaInput {
  /** Latitude in degrees, +N. */
  lat: number;
  /** Longitude in degrees, +E. */
  lon: number;
  /** Course Over Ground, radians (0 = N, π/2 = E). */
  cog: number;
  /** Speed Over Ground, m/s. */
  sog: number;
}

export interface CpaResult {
  /** Closest predicted distance, meters. */
  cpaMeters: number;
  /** Seconds until that closest distance; negative means it already happened. */
  tcpaSeconds: number;
  /** Current straight-line range, meters. */
  rangeMeters: number;
  /** Current bearing from own to target, radians (compass). */
  bearingRadians: number;
}

const M_PER_DEG_LAT = 111_320;

export function computeCpa(own: CpaInput, target: CpaInput): CpaResult {
  const ownLatRad = (own.lat * Math.PI) / 180;
  const mPerDegLon = M_PER_DEG_LAT * Math.cos(ownLatRad);

  const rx = (target.lon - own.lon) * mPerDegLon;
  const ry = (target.lat - own.lat) * M_PER_DEG_LAT;

  const ownVx = own.sog * Math.sin(own.cog);
  const ownVy = own.sog * Math.cos(own.cog);
  const tgtVx = target.sog * Math.sin(target.cog);
  const tgtVy = target.sog * Math.cos(target.cog);

  const rvx = tgtVx - ownVx;
  const rvy = tgtVy - ownVy;
  const relSpeedSq = rvx * rvx + rvy * rvy;

  let tcpa: number;
  if (relSpeedSq < 1e-9) {
    tcpa = 0;
  } else {
    tcpa = -(rx * rvx + ry * rvy) / relSpeedSq;
  }

  const cpaX = rx + tcpa * rvx;
  const cpaY = ry + tcpa * rvy;
  const cpa = Math.sqrt(cpaX * cpaX + cpaY * cpaY);

  const range = Math.sqrt(rx * rx + ry * ry);
  // Bearing from own to target — compass radians (0 = N, +ve = E)
  const bearing = Math.atan2(rx, ry);
  const bearingWrapped = (bearing + 2 * Math.PI) % (2 * Math.PI);

  return {
    cpaMeters: cpa,
    tcpaSeconds: tcpa,
    rangeMeters: range,
    bearingRadians: bearingWrapped,
  };
}
```

```ts
// packages/compute/src/ais/cpa.test.ts
import { describe, it, expect } from 'vitest';
import { computeCpa } from './cpa.js';

const KN = 0.514444;
const NM_TO_M = 1852;
const DEG = Math.PI / 180;

describe('computeCpa', () => {
  it('head-on collision: 1 NM apart, both 10 kn straight at each other → tcpa ≈ 180s, cpa ≈ 0', () => {
    const own = { lat: 0, lon: 0, cog: 0, sog: 10 * KN }; // heading N
    // Target 1 NM north, heading S at 10 kn.
    const target = {
      lat: 1 / 60, // 1 NM ≈ 1/60 degree latitude
      lon: 0,
      cog: Math.PI, // heading S
      sog: 10 * KN,
    };
    const r = computeCpa(own, target);
    expect(r.rangeMeters).toBeCloseTo(NM_TO_M, -1); // 1 NM
    expect(r.tcpaSeconds).toBeCloseTo(180, 0); // closing at 20 kn ≈ 1 NM in 3 min
    expect(r.cpaMeters).toBeLessThan(10); // basically zero
    expect(r.bearingRadians).toBeCloseTo(0, 2); // due north
  });

  it('parallel courses: same speed same heading → never converges, cpa = current range', () => {
    const own = { lat: 0, lon: 0, cog: 0, sog: 5 * KN };
    const target = { lat: 0, lon: 0.01, cog: 0, sog: 5 * KN };
    const r = computeCpa(own, target);
    expect(r.cpaMeters).toBeCloseTo(r.rangeMeters, 0); // doesn't get closer
    expect(Math.abs(r.tcpaSeconds)).toBeLessThan(1); // tcpa ≈ 0 (no relative motion)
  });

  it('diverging: own going N, target going S behind, tcpa negative', () => {
    const own = { lat: 0, lon: 0, cog: 0, sog: 5 * KN };
    const target = { lat: -1 / 60, lon: 0, cog: Math.PI, sog: 5 * KN };
    const r = computeCpa(own, target);
    expect(r.tcpaSeconds).toBeLessThan(0);

  it('handles zero relative motion (same vel) without dividing by zero', () => {
    const own = { lat: 45, lon: -75, cog: Math.PI / 4, sog: 7 };

    expect(Number.isFinite(r.tcpaSeconds)).toBe(true);
  });

    const own = { lat: 0, lon: 0, cog: 0, sog: 0 };
```

```ts
export async function PUT(req: Request): Promise<Response> {
  const store = getSharedConfigStore();
  if (!store) return Response.json({ error: 'ConfigStore not initialised' }, { status: 503 });
  const body = (await req.json()) as Partial<AisAlarmConfig>;
  // Validate
  const c = { ...store.aisAlarmConfig$.value, ...body };
  if (typeof c.enabled !== 'boolean')
    return Response.json({ error: 'enabled must be boolean' }, { status: 400 });
  if (!Number.isFinite(c.cpaMeters) || c.cpaMeters <= 0)
    return Response.json({ error: 'cpaMeters must be positive' }, { status: 400 });
  if (!Number.isFinite(c.tcpaSeconds) || c.tcpaSeconds <= 0)
    return Response.json({ error: 'tcpaSeconds must be positive' }, { status: 400 });
  await store.setAisAlarmConfig(c);
  return Response.json(c);
}
```

### Task 5: `/chart` page (SVG canvas + AIS overlay + CPA alarm UI)

**Files:**

- Create: `packages/web/src/app/chart/page.tsx`
- Modify: `packages/web/src/app/Navbar.tsx` — add `{ href: '/chart', label: 'Chart' }` between `/helm` and `/polars`

#### Behaviour

- 600×600 (or aspect-ratio-fit) SVG canvas.
- Own boat at center, pointing up (course-up).
- Range rings: 1, 2, 4, 8 NM by default. Configurable via dropdown.
- North arrow indicator in a corner (rotates relative to current heading).
- AIS targets rendered as small triangles, oriented by COG, position relative to own.
- Leader line per target: line from target's position to where it'll be in 6 minutes (configurable). Length = sog \* 6min.
- Targets with CPA below `cpaMeters` AND TCPA between 0 and `tcpaSeconds` are highlighted red with a pulsing ring.
- Click target → side panel with MMSI, name, COG (deg), SOG (kn), CPA (NM), TCPA (mm:ss).
- Top bar: range selector, north-up / course-up toggle, AIS alarm config (enable, CPA threshold, TCPA threshold).
- Poll `/api/ais/targets` every 2 s. Subscribe to SSE stream for own-boat updates.

#### Skeleton

```tsx
'use client';

import { useEffect, useMemo, useRef, useState } from 'react';
import { computeCpa, type CpaResult } from '@g5000/compute';
import type { AisTarget } from '@g5000/core';
import { useSse } from '../../hooks/use-sse';

const NM = 1852;
const KN_TO_MS = 0.514444;
const RANGE_OPTIONS_NM = [1, 2, 4, 8, 16];

interface AisAlarmConfig {
  enabled: boolean;
  cpaMeters: number;
  tcpaSeconds: number;
}

export default function ChartPage() {
  const { channels } = useSse();
  const [targets, setTargets] = useState<AisTarget[]>([]);
  const [alarmConfig, setAlarmConfig] = useState<AisAlarmConfig>({
    enabled: true,
    cpaMeters: NM,
    tcpaSeconds: 600,
  });
  const [rangeNm, setRangeNm] = useState(4);
  const [northUp, setNorthUp] = useState(false);
  const [selectedMmsi, setSelectedMmsi] = useState<number | null>(null);

  // Poll AIS targets every 2s
  useEffect(() => {
    const fetchTargets = async () => {
      const r = await fetch('/api/ais/targets', { cache: 'no-store' });
      if (r.ok) {
        const j = (await r.json()) as { targets: AisTarget[] };
        setTargets(j.targets);
      }
    };
    fetchTargets();
    const id = setInterval(fetchTargets, 2000);
    return () => clearInterval(id);
  }, []);

  // Load alarm config
  useEffect(() => {
    fetch('/api/ais/alarm-config')
      .then((r) => r.ok && r.json())
      .then((c) => c && setAlarmConfig(c));
  }, []);

  // Own boat state from SSE
  const ownLat = channels.get('nav.gps.position')?.value?.value?.lat ?? null;
  const ownLon = channels.get('nav.gps.position')?.value?.value?.lon ?? null;
  const ownCog = channels.get('nav.gps.cog')?.value?.value ?? 0;
  const ownSog = channels.get('nav.gps.sog')?.value?.value ?? 0;
  const ownHeading = channels.get('boat.heading.magnetic')?.value?.value ?? ownCog;

  // Compute CPA per target (memoized on inputs)
  const targetsWithCpa = useMemo(() => {
    if (ownLat === null || ownLon === null) return [];
    return targets.map((t) => {
      if (t.lat === undefined || t.lon === undefined) return { target: t, cpa: null };
      const own = { lat: ownLat, lon: ownLon, cog: ownCog, sog: ownSog };
      const tgt = { lat: t.lat, lon: t.lon, cog: t.cog ?? 0, sog: t.sog ?? 0 };
      return { target: t, cpa: computeCpa(own, tgt) };
    });
  }, [targets, ownLat, ownLon, ownCog, ownSog]);

  // SVG rendering: 600×600, scale so rangeNm * NM maps to 280 px (radius)
  const svgSize = 600;
  const svgRadius = 280;
  const metersToPx = svgRadius / (rangeNm * NM);

  const isThreat = (cpa: CpaResult | null): boolean =>
    !!cpa &&
    alarmConfig.enabled &&
    cpa.cpaMeters < alarmConfig.cpaMeters &&
    cpa.tcpaSeconds > 0 &&
    cpa.tcpaSeconds < alarmConfig.tcpaSeconds;

  return (
    <main className="p-6 max-w-7xl mx-auto text-slate-100">
      <div className="flex items-center justify-between mb-4">
        <h1 className="text-2xl font-semibold">Chart</h1>
        <div className="flex gap-3 items-center text-sm font-mono">
          <label>
            Range:&nbsp;
            <select
              value={rangeNm}
              onChange={(e) => setRangeNm(Number(e.target.value))}
              className="bg-slate-800 px-2 py-1 rounded"
            >
              {RANGE_OPTIONS_NM.map((n) => (
                <option key={n} value={n}>
                  {n} NM
                </option>
              ))}
            </select>
          </label>
          <label>
            <input
              type="checkbox"
              checked={northUp}
              onChange={(e) => setNorthUp(e.target.checked)}
            />
            &nbsp;North up
          </label>
          <button
            onClick={async () => {
              const next = { ...alarmConfig, enabled: !alarmConfig.enabled };
              await fetch('/api/ais/alarm-config', {
                method: 'PUT',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify(next),
              });
              setAlarmConfig(next);
            }}
            className={`px-2 py-1 rounded text-xs ${alarmConfig.enabled ? 'bg-emerald-700' : 'bg-slate-700'}`}
          >
            Alarm {alarmConfig.enabled ? 'ON' : 'OFF'}
          </button>
        </div>
      </div>

      <div className="flex gap-4">
        <svg
          width={svgSize}
          height={svgSize}
          viewBox={`0 0 ${svgSize} ${svgSize}`}
          className="bg-slate-950 border border-slate-800 rounded"
        >
          {/* Range rings */}
          {[1, 2, 4]
            .filter((r) => r * NM * metersToPx < svgRadius)
            .map((r) => (
              <circle
                key={r}
                cx={svgSize / 2}
                cy={svgSize / 2}
                r={r * NM * metersToPx}
                fill="none"
                stroke="#334155"
                strokeDasharray="4 4"
              />
            ))}
          {/* Outer ring at full range */}
          <circle cx={svgSize / 2} cy={svgSize / 2} r={svgRadius} fill="none" stroke="#475569" />

          {/* Own boat — triangle pointing up */}
          <g
            transform={`translate(${svgSize / 2}, ${svgSize / 2}) rotate(${northUp ? (-ownHeading * 180) / Math.PI : 0})`}
          >
            <polygon points="0,-12 -7,8 7,8" fill="#fbbf24" stroke="#0f172a" strokeWidth="1" />
          </g>

          {/* AIS targets */}
          {targetsWithCpa.map(({ target, cpa }) => {
            if (!cpa) return null;
            const angle = cpa.bearingRadians - (northUp ? 0 : ownHeading);
            const dist = cpa.rangeMeters * metersToPx;
            if (dist > svgRadius) return null; // off-screen
            const x = svgSize / 2 + dist * Math.sin(angle);
            const y = svgSize / 2 - dist * Math.cos(angle);
            const threat = isThreat(cpa);
            const targetCogRel = (target.cog ?? 0) - (northUp ? 0 : ownHeading);
            const leaderMin = (6 / 60) * 3600; // 6 min in s
            const leaderLen = (target.sog ?? 0) * leaderMin * metersToPx;
            return (
              <g
                key={target.mmsi}
                onClick={() => setSelectedMmsi(target.mmsi)}
                style={{ cursor: 'pointer' }}
              >
                <g transform={`translate(${x}, ${y}) rotate(${(targetCogRel * 180) / Math.PI})`}>
                  <polygon
                    points="0,-7 -4,5 4,5"
                    fill={threat ? '#ef4444' : '#94a3b8'}
                    stroke="#0f172a"
                    strokeWidth="0.5"
                  />
                  {leaderLen > 1 && (
                    <line
                      x1="0"
                      y1="-7"
                      x2="0"
                      y2={-7 - leaderLen}
                      stroke={threat ? '#ef4444' : '#475569'}
                      strokeWidth="1"
                    />
                  )}
                </g>
                {threat && (
                  <circle cx={x} cy={y} r="14" fill="none" stroke="#ef4444" strokeWidth="2">
                    <animate
                      attributeName="r"
                      values="10;18;10"
                      dur="2s"
                      repeatCount="indefinite"
                    />
                    <animate
                      attributeName="opacity"
                      values="1;0;1"
                      dur="2s"
                      repeatCount="indefinite"
                    />
                  </circle>
                )}
              </g>
            );
          })}
        </svg>

        {/* Side panel */}
        <div className="flex-1">
          <h2 className="font-semibold mb-2">Targets ({targetsWithCpa.length})</h2>
          <table className="w-full text-xs font-mono">
            <thead>
              <tr className="text-slate-400 border-b border-slate-800">
                <th className="text-left">MMSI</th>
                <th className="text-left">Name</th>
                <th className="text-right">Range</th>
                <th className="text-right">CPA</th>
                <th className="text-right">TCPA</th>
              </tr>
            </thead>
            <tbody>
              {targetsWithCpa
                .filter(({ cpa }) => cpa && cpa.rangeMeters < rangeNm * NM)
                .sort((a, b) => (a.cpa?.cpaMeters ?? 1e9) - (b.cpa?.cpaMeters ?? 1e9))
                .map(({ target, cpa }) => (
                  <tr
                    key={target.mmsi}
                    className={`border-b border-slate-900 ${selectedMmsi === target.mmsi ? 'bg-slate-800' : ''} ${isThreat(cpa) ? 'text-red-300' : ''}`}
                    onClick={() => setSelectedMmsi(target.mmsi)}
                  >
                    <td className="py-1">{target.mmsi}</td>
                    <td className="py-1">{target.name ?? '—'}</td>
                    <td className="py-1 text-right">
                      {cpa ? (cpa.rangeMeters / NM).toFixed(2) : '—'} NM
                    </td>
                    <td className="py-1 text-right">
                      {cpa ? (cpa.cpaMeters / NM).toFixed(2) : '—'} NM
                    </td>
                    <td className="py-1 text-right">
                      {cpa && cpa.tcpaSeconds > 0
                        ? `${Math.floor(cpa.tcpaSeconds / 60)}:${String(Math.floor(cpa.tcpaSeconds % 60)).padStart(2, '0')}`
                        : '—'}
                    </td>
                  </tr>
                ))}
            </tbody>
          </table>
        </div>
      </div>
    </main>
  );
}
```
